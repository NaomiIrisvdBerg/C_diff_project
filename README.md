# C_diff_project
Understanding ecology and mechanisms driving the post-antibiotics C. diff resistance of a synthetic gut community, found by the Joy Scaria Lab.
In this repository, you can find R scripts for:
A) data (statistical) analyses, 
B) mathematical modelling of Mix-14 interaction dynamics.
